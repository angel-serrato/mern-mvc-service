# Frontend — client/

Aplicación React para la gestión de productos. Utiliza Vite, TypeScript y TailwindCSS para una experiencia moderna y rápida.

## Estructura

- `src/components/` — Componentes principales (Layout, ProductForm, ProductList, ProductItem)
- `src/services/` — Lógica para consumir la API REST del backend
- `src/config/` — Configuración de Axios
- `public/` — Archivos estáticos

## Scripts

```bash
npm install      # Instala dependencias
npm run dev      # Inicia el servidor de desarrollo
npm run build    # Compila para producción
```

## Características

- Listado, creación, edición y eliminación de productos
- Consumo de API REST
- Manejo de errores y estados
- Estilos con TailwindCSS

## Requisitos

- Node.js >= 18
- Backend corriendo en `/api/products`
